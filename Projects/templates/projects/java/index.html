Java was the first compiled programming language that I started to learn on my own. Originally, I wanted to use it to mod the game Minecraft. However it opened the doors to several different courses as well as a deeper understanding of object oriented programming.

To learn Java, I signed up for an “Intro To Java Programming” course offered by San Jose State University on Udacity. After completing the course, I instantly started diving into modding Minecraft. Unfortunately, through the shuffle of upgrading computers, the first mod I worked on was completely lost, with the exception of a single screenshot.

After playing around with Minecraft a bit, I wanted to expand my knowledge of programming. With Java as the catalyst, I was able to take a course on Udemy that provided my first exposure to data structures and algorithms. This course was quickly followed up by second course in algorithms offered by Robert Sedgewick.

The last course I took on Java was actually an introduction to Android Development. Those projects are grouped under a different section.
